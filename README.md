# Sort Algorithms

## Bubble Sort
<details>
  <summary>Complexity</summary><br/>

  > Time Complexity
  - Best Case = O(n)
  - Average Case = O(n²)
  - Worst Case = O(n²)

  > Space Complexity
  - O(1)
</details>

<details>
  <summary>Execution Time</summary><br/>

  > 100.000 Registers
  - ~22 seconds (~0.36 minutes)

  > 100.000.000 Registers
  - ~2266 seconds (~37 minutes)
</details>
